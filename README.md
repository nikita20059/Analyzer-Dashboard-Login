# Analyzer Dashboard Login Page

## Project Description

Analyzer Dashboard Login Page is a modern and responsive authentication interface designed for a server monitoring and analysis dashboard.
The system allows administrators or users to securely log in and access the dashboard for monitoring server performance and system analytics.

This project focuses on building a clean UI login page using the **MERN Stack**.

---

## Technologies Used

* React JS
* Node.js
* Express.js
* MongoDB
* CSS3
* REST API

---

## Features

* Secure Login Interface
* Responsive UI Design
* Modern Dashboard Style Layout
* Clean and Professional Login Card
* Background Design with Visual Elements
* Ready for Backend Authentication Integration

---

## UI Design Overview

### Header Section

* Background image or gradient
* Company / system logo
* Clean and professional layout

### Login Card

* Email / Username input field
* Password input field
* Login button
* Modern card design with shadow

### Background Design

* Stylish dashboard theme
* Light grid / modern UI background
* Responsive layout for all screen sizes

## Installation

Clone the repository

```
git clone https://github.com/yourusername/analyzer-dashboard-login.git
```

---

### Install Frontend

```
cd frontend
npm install
npm start
```

---

### Install Backend

```
cd backend
npm install
node server.js
```
